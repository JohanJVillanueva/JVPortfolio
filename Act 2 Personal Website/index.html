Designed and prototyped a working 11-key macropad with knob using the Arduino Platform, Cherry MX Switches and C Programming language. Using common materials found in makerlab as well as the use of an atmega32u4, I was able to make an affordable macropad.

Designed, and developed a mobile application that highlights the history, culture and tradition of Cavite City. The application was developed using HTML, CSS and JQUERY and converted to “. APK”.

Development of Caviteñan Mobile Application – 2018

Designed, and developed a mobile application that disseminated timely information to the students of Cavite National High School. The main objective of the application is to be able to eradicate disinformation in the school while providing DRRM Safety infographics. The application was developed using HTML, CSS and JQUERY and converted to “. APK”.
